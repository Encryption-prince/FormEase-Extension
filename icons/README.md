# Extension Icons

This directory contains the extension icons in various sizes:
- icon16.png (16x16) - Small icon for extension management
- icon32.png (32x32) - Medium icon for extension management  
- icon48.png (48x48) - Extension management and Chrome Web Store
- icon128.png (128x128) - Chrome Web Store and installation

Note: Actual icon files need to be created as PNG images. These are referenced in manifest.json.
